# Лабораторная работа №4: Десктопное приложение на Windows Forms

Данный репозиторий содержит выполнение лабораторной работы по созданию десктопного приложения на Windows Forms и освоению навыков структурирования проекта. Работа включает две части: изучение механизмов ООП в C++ и портирование консольного приложения в графический интерфейс.

В первой части реализованы упражнения по множественному наследованию и решению проблемы ромба, перегрузке операторов на примере класса вектора, передаче объектов в качестве аргументов функций, а также выполнено разделение консольного проекта на многофайловую структуру с использованием заголовочных и исходных файлов.

Во второй части консольное приложение системы грузоперевозок портировано в Windows Forms. Приложение поддерживает два окна с перегрузкой конструктора, загрузку и сохранение данных, добавление, редактирование и удаление записей с визуальным обновлением в DataGridView, сортировку и валидацию ввода. Классы данных вынесены в отдельные файлы, использованы стандартные компоненты OpenFileDialog и DataGridView.

Проект полностью компилируется в исполняемый файл и готов к запуску в среде Windows. Код снабжён комментариями, структура решения соответствует стандартам организации многофайловых проектов в Visual Studio.
